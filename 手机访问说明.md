# 手机访问人脸签到系统指南

## 方法一：使用本地服务器（推荐）

### 步骤1：启动本地服务器

1. 在电脑上打开命令提示符（Win+R，输入cmd）或PowerShell
2. 切换到项目目录：
   ```
   cd C:/Users/86157/Desktop/人脸识别签到
   ```
3. 运行服务器：
   ```
   python start_server.py
   ```
4. 服务器启动后，您会看到类似以下信息：
   ```
   服务器启动成功！
   本地访问: http://localhost:8000
   手机访问: http://192.168.1.100:8000
   按Ctrl+C停止服务器
   ```

### 步骤2：确保网络连接

1. 确保您的电脑和手机连接到同一个WiFi网络
2. 如果是公司或学校网络，可能需要确认设备间可以互相访问

### 步骤3：在手机上访问

1. 在手机浏览器中输入显示的"手机访问"地址（例如：http://192.168.1.100:8000）
2. 点击打开2.html文件
3. 允许浏览器访问摄像头（如果提示）

## 方法二：使用在线托管平台

1. 将2.html文件上传到以下任一平台：
   - GitHub Pages
   - Netlify
   - Vercel
   - CodePen
2. 获取生成的链接
3. 在手机浏览器中访问该链接

## 方法三：通过文件传输应用

1. 使用微信、QQ或其他文件传输应用将2.html发送到手机
2. 在手机上使用Chrome或Firefox浏览器打开文件
3. 可能需要在浏览器设置中允许访问本地文件

## 注意事项

1. **摄像头权限**：
   - 首次使用时，浏览器会请求摄像头权限
   - 请确保点击"允许"

2. **HTTPS要求**：
   - 某些浏览器要求HTTPS环境才能访问摄像头
   - 如果遇到问题，可以尝试：
     - 使用Chrome浏览器
     - 在地址栏输入 `chrome://flags/`
     - 搜索"unsafely-treat-insecure-origin-as-secure"
     - 添加您的本地IP地址并启用

3. **模型加载**：
   - 首次使用需要下载人脸识别模型（约10MB）
   - 请确保网络连接稳定
   - 如果加载失败，可以尝试手动下载模型文件到本地models文件夹

4. **手机操作提示**：
   - 点击"切换摄像头"按钮可在前置和后置摄像头间切换
   - 建议使用后置摄像头进行人脸识别，效果更好
   - 录入人脸时，建议从多个角度录入以提高识别率

## 故障排除

1. **无法访问服务器**：
   - 检查电脑和手机是否在同一网络
   - 确认Windows防火墙没有阻止Python
   - 尝试关闭电脑上的VPN

2. **摄像头无法工作**：
   - 检查浏览器权限设置
   - 尝试使用Chrome或Firefox浏览器
   - 确认手机摄像头没有被其他应用占用

3. **模型加载失败**：
   - 尝试刷新页面
   - 检查网络连接
   - 考虑手动下载模型文件

如有其他问题，请检查浏览器控制台（通常在设置-开发者工具中）的错误信息。
